# PromptForge Core Features

## 1. Prompt Library
- Create, read, update, and delete prompts.
- Organize prompts using tags and favorites.
- Advanced search, filtering, and sorting capabilities.

## 2. Prompt Inbox
- A fast capture area for raw prompt ideas.
- Seamlessly convert inbox items into fully structured prompts.

## 3. Context Packs
- Create and manage reusable chunks of context (e.g., style guidelines, background knowledge, code snippets).
- Inject context packs into any prompt during the compilation phase.

## 4. Prompt Variables & Metadata
- Define dynamic placeholders (variables) within prompt templates (e.g., `{{variable_name}}`).
- Add rich metadata to variables to describe their purpose and expected types.

## 5. Prompt Compiler
- A dedicated workspace to resolve variables and inject context packs.
- Generates the final, ready-to-use prompt text.

## 6. Prompt Examples
- Save test runs and example outputs directly alongside the prompt.
- Helpful for tracking prompt performance across different inputs.

## 7. Multi-LLM Output Comparison
- Generate and compare outputs from multiple LLMs side-by-side.
- Helps identify the best prompt formulation for specific models.

## 8. Import/Export v0
- Export the complete local database (vault) to a single JSON file.
- Import a previously exported JSON file to restore or transfer data.
